This Is My Semester VI Project -  Cervical spondylosis prediction using Machine Learning

Cervical spondylosis is a common degenerative spine condition causing neck pain, stiffness, and neurological symptoms, primarily affecting older adults. Early detection of cervical spondylosis is crucial for effective treatment, but traditional diagnostic methods can be time-consuming and subjective. Machine learning has the potential to enhance diagnostic accuracy by analyzing complex medical data and predicting conditions like cervical spondylosis. The system utilizes various health-related features such as age, neck pain, and stiffness to predict the likelihood of cervical spondylosis, helping clinicians identify high-risk patients.


Developed ML model like random forest, Decision Tree, SKlearn using symptoms & lifestyle data

Result -    The Random Forest Classifier effectively predicts cervical spondylosis outcomes using features like age, neck pain, and stiffness. 

These features are critical for diagnosis and offer valuable insights. 

The system shows potential for integration into healthcare tools, aiding early diagnosis and better management of the condition.
